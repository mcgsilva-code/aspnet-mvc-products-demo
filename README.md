# ASP.NET MVC Products Demo

## Descrição
Projeto de demonstração em **ASP.NET MVC** com `ProductsController` e `List View`.  
Feito para portfólio e entrevistas, mostrando conhecimento em C#, .NET e MVC.

## Tecnologias
- ASP.NET MVC
- C#
- .NET
- Visual Studio

## Como rodar
1. Abra a solução `AspNetWebApp.sln` no Visual Studio.
2. Clique em **Run (Ctrl + F5)**.
3. Navegue até `/Products/List` para ver a lista de produtos.
